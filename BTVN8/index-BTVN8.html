Coersion - Truthy and Falsy

1. Kết quả khi console.log(a,b,c); là 10 10 10. Chứng minh giống nhau về giá trị và dữ liệu (Numeric)

- Theo như em hiểu thì coercion là hành động chuyển dữ liệu.
+ Explicit coercion được thể hiện qua các hàm (ParseInt(), Number(), Stringify()...) tường minh
+ Implicit coercion thường gặp trong biểu thức điều kiện của if, while, trong loop và trường hợp so sánh tương đối (==)

2. Khi một giá trị không phải boolean cần chuyển sang boolean sẽ cho giá trị True ngoại trừ các trường hợp

  • 0
  • Null
  • Undefined
  • Chuỗi rỗng
  • NaN
  • False

- Ví dụ đưa ra log ra kết quả "a is truthy"

Logical Operators and Conditional

1. q: True; w: False(chuỗi rỗng); || chỉ cần 1 kết quả True => Kết quả trả về True

2. e: True; r: False(chuỗi rỗng); !r: True; && cả 2 kết quả True => Kết quả trả về True

3. Tạo một câu điều kiện if sử dụng toán tử logic && để kết quả trả về true
